## InvalidAggregationStrategyException

```python
InvalidAggregationStrategyException(*args, **kwargs)
```

Common base class for all non-exit exceptions related to invalid aggregation strategies.

### Inheritance

**Inheritance Chain**: 
- `InvalidAggregationStrategyException` 
- → `ValidationException`
- → `Exception` 
- → `BaseException`
